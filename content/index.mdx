QUE HACE?

la variable rnd crea una instancia de Random, esto servira para crear numero aleatorios.

```java
Random rnd = new Random();
```

Son tres ciclos `for`.

- El primero va a repetirse el numero de tablero que pidamos en este caso 75.
- El segundo se repetira la cantidad de columnas del tablero.
- El tercero se repetira la cantidad de filas del tablero, ademas de aqui estar la logica de los numeros aleatorios.

```java
if ( j == 0) {
  tableroBingo[i][j] = rnd.nextInt(15-1+1)+1;
}else if ( j == 1) {
  tableroBingo[i][j] = rnd.nextInt(30-15+1)+15;
}else if ( j == 2) {
  tableroBingo[i][j] = rnd.nextInt(45-30+1)+30;
}else if ( j == 3) {
  tableroBingo[i][j] = rnd.nextInt(60-45+1)+45;
}else if ( j == 4) {
  tableroBingo[i][j] = rnd.nextInt(75-60+1)+60;
}
```

Aqui se hara un `if` anidado para establecer los rangos de los numeros, esto por que en el bingo del proyecto final dice que dependiendo la columna el rango de numeros varia.

esta declaracion de variable lo que hace es establecer el numero dentro de una cacilla donde i es la columna y j la fila,

```java
tableroBingo[i][j]
```

este pedazo de codigo genera el numero aleatorio dentro del rango que le asignemos.

```java
rnd.nextInt(numeroMaximo-numeroMinimo+1)+numeroMaximo;
```

donde `numeroMaximo` es el valor maximo que puede alcanzar el numero y `numeroMinimo` el minimo valor que este puede tener.
Ejemplo:
para crear un numero aleatorio en un rango de 30 a 45, el codigo seria asi:

```java
  tableroBingo[i][j] = rnd.nextInt(45-30+1)+30;
```


`button` se remplaza con la variable que contiene el boton o cualquiero otro elemento, este codigo se pone al final de la funcion principal.
